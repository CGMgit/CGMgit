### 🔭 研究兴趣

- 分析哲学、逻辑模型与基础数学
- 概率论与测度论、非参数统计、贝叶斯推断  
- 算子理论视角下的优化与控制（KKT / ADMM / OSQP）  
- 神经网络与强化学习在最优控制中的应用  
- 自适应 MPC（AN-MPC）与路径规划（MD-A*）算法设计  

🧠 当前研究主题：ZFC公理系统与范畴理论
